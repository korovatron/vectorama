How to Use Vectorama

Getting Started

Use Add to insert vectors, lines, planes, and matrices.
Switch between 2D and 3D with the dimension toggle.
Use Reset Axes to quickly restore a clean view.

Navigation

Pan/rotate/zoom with mouse and wheel.
Pan with one finger and zoom with pinch.
Use + / - keyboard keys for zoom.
Use arrow keys to pan the camera.

Objects & Analysis

Edit object coordinates directly in the panel rows.
Lines and planes support multiple equation forms (e.g. Cartesian, vector, scalar/dot). Use each row’s form toggle button to switch formats.
Toggle visibility with the color square beside each object.
Use the i buttons for lengths, angles, and distances.
Use bottom toggles for grid and intersection visuals.

Matrices

Select or edit a matrix, then press the play button to apply it.
Open matrix info to inspect eigenvalues and eigenspaces.
Tap/click outside this panel to close
